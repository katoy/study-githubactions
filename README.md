# NumberHolder プロジェクト

NumberHolder プロジェクトは、2つの整数を保持し、それらの整数を返すシンプルな Python クラスです。

## 必要な環境

*   Python 3.9 以降
*   pytest

## インストール方法

```bash
pip install -r requirements.txt
```

## 実行方法

```bash
python number_class.py
```

## テスト方法

```bash
pytest
```

## 貢献方法

1.  リポジトリをフォークします。
2.  新しいブランチを作成します。
3.  変更をコミットします。
4.  プルリクエストを送信します。
